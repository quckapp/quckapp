# Event Broadcast Service

Elixir service for SSE/WebSocket event distribution.

## Overview

| Property | Value |
|----------|-------|
| **Port** | 4006 |
| **Database** | MongoDB |
| **Framework** | Phoenix 1.7 |
| **Language** | Elixir 1.15 |

## Features

- Server-Sent Events (SSE)
- WebSocket broadcasting
- Event filtering
- Fan-out distribution
