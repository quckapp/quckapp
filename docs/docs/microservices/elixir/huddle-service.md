# Huddle Service

Elixir service for audio rooms and live discussions.

## Overview

| Property | Value |
|----------|-------|
| **Port** | 4005 |
| **Database** | MongoDB |
| **Framework** | Phoenix 1.7 |
| **Language** | Elixir 1.15 |

## Features

- Audio-only rooms
- Speaker management
- Hand raising
- Room scheduling
- Recording support
