# File Service

Go service for file storage and management.

## Overview

| Property | Value |
|----------|-------|
| **Port** | 5002 |
| **Database** | MongoDB |
| **Framework** | Gin |
| **Language** | Go 1.21 |

## Features

- Multi-part uploads
- S3/Azure Blob integration
- Virus scanning
- File versioning
- Access control
