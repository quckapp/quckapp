# QuckChat Features

This section documents all the features implemented in QuckChat, including both core messaging functionality and advanced features.

## Core Features

- **User Management** - Registration, authentication, profile management
- **Conversations** - One-on-one and group messaging
- **Messages** - Text, media, attachments, reactions
- **Real-time** - WebSocket-based instant messaging
- **Push Notifications** - FCM and Expo notifications

## Advanced Features

### Broadcast Lists

Send messages to multiple contacts without creating a group. Recipients receive messages as individual chats and can't see other recipients.

[Learn more about Broadcast Lists](/features/new-features#1--broadcast-lists)

### Linked Devices

Use QuckChat on multiple devices simultaneously. All messages sync in real-time across all linked devices.

[Learn more about Linked Devices](/features/new-features#2--linked-devices)

### Communities

Create communities to organize multiple groups with shared membership. Similar to WhatsApp Communities.

[Learn more about Communities](/features/new-features#3-%EF%B8%8F-communities)

### Starred Messages

Star important messages to find them quickly later. Works across all conversations.

[Learn more about Starred Messages](/features/new-features#4--starred-messages)

### Read All

Mark all unread conversations as read with a single tap.

[Learn more about Read All](/features/new-features#5--read-all)
